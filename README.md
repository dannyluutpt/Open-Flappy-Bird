# FlappieBus 🚌

A Flappy Bird clone where the bird is a **winged flying bus**. Tap / click / press Space to fly. Works on desktop and mobile.

## Features
- Plays on PC (click / Space) and phone (tap)
- Settings: pick **bus color** (8 colors) and **background** (Day / Sunset / Night / Space)
- High score saved locally
- Single self-contained `index.html`, no build step, no dependencies

## Run locally
Just open `index.html` in a browser.
